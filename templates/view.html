Persephone - visualise my hash

The application visually represents hexadecimal code as used within hashes in the form of stacked square blocks. If you enter anything except the hexadecimal characters (0123456789abcdef), the application will return a 128-bit hash generated using hades. Each single vertical line represents a number from 0 to 255, i.e. 00 to ff. A completely white vertical line would represent 00, while a completely black line would represent ff.

To directly use the image, use the endpoint, /api/:hash (e.g. /api/00007e10107e007e0000).
